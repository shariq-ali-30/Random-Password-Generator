# Random Password Generator

## About The Project

This project is a Random Password Generator built using HTML, CSS, and JavaScript.

The purpose of this assignment was to practice JavaScript fundamentals, DOM manipulation, event handling, random number generation, and creating a simple utility application that generates secure random passwords with a single button click.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Features

- Generate random passwords with a single click
- 8-character password generation
- Combination of uppercase letters, lowercase letters, numbers, and special characters
- Clean and minimal user interface
- Responsive design for different screen sizes
- Interactive button with click animation
- Dynamic password display using DOM manipulation

## What I Learned

Through this assignment, I learned:

- How to generate random values using JavaScript
- How to use `Math.random()` and `Math.floor()` effectively
- How to manipulate the DOM using JavaScript
- How to handle button click events
- How to build a simple utility application
- How to combine HTML, CSS, and JavaScript to create an interactive project
- How to improve user experience with a clean and responsive interface
